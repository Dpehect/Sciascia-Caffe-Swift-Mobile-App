# Sciascia Caffè 1919 - iOS Companion Application

A luxurious, high-contrast, and technologically advanced iOS companion application for **Sciascia Caffè 1919** (Rome, Italy). Built entirely using modern Swift, SwiftUI, SwiftData, and RealityKit.

## Core Features

- **Interactive AR Menu:** Points the device camera at any flat surface to project floatable 3D virtual coffee cups, pastry selections, allergen disclosures, nutritional details, and ingredients folders. Includes simulated 3D model interaction mechanics on macOS platforms.
- **Home Dashboard:** Luxurious welcomes, quick reordering logs, and active loyalty stamps status cards.
- **Loyalty Stamp Program:** Animated 10-stamp loyalty program grid cards using custom spring animations that increment stamp counts automatically upon adisyon checkout completion.
- **Biometric Security Protection:** Face ID and Touch ID credentials gatekeepers to protect staff stock inventory configurations.
- **Order Tracking List:** Real-time adisyon queue monitors, status badges (Preparing, Ready, Served, Cancelled), and quick swiping gestures.
- **High-Contrast Typography Design:** Tailored light-mode cream layouts featuring Deep Brown (`#2C2118`) and Dark Gray (`#44403C`) typography alignments for perfect text contrast and reading accessibility.

## Technology Stack

- **Frameworks:** SwiftUI, RealityKit, ARKit, LocalAuthentication, TipKit, Swift Charts, AppIntents.
- **Database Engine:** SwiftData (Offline-First architecture with local migrations).
- **Target OS:** iOS 17.0+ (Universal compilation support with macOS compatibility fallbacks).
